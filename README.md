# Push-up-Counter-using-Real-time-Pose-Estimation
Developed an AI-powered Push-Up Counter that detects and counts push-ups in real time using computer vision and pose estimation techniques. The project uses a webcam feed to track body movements and analyze joint angles to determine whether a push-up repetition is completed correctly. Built using Python, OpenCV, and MediaPipe, the system provides accurate repetition counting along with live visual feedback on posture and movement.

This project demonstrates practical applications of machine learning and computer vision in fitness tracking, human pose detection, and real-time motion analysis. It also helped strengthen skills in Python programming, image processing, and AI-based automation.

## Features

- Real-time push-up detection
- Accurate repetition counting
- Live webcam tracking
- Pose estimation using MediaPipe
- Visual feedback during exercise

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
